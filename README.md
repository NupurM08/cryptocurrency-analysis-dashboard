# 📊 Cryptocurrency Dashboard

## 📌 Objective
To create an interactive dashboard for analyzing cryptocurrency market trends.

## 🛠️ Tools Used
- Power BI

## 📊 Dataset
The dataset contains cryptocurrency data such as price, market trends, and other related metrics.

## 📈 Features
- Price trend analysis of cryptocurrencies
- Comparison of top cryptocurrencies
- Interactive filters and slicers for better analysis

## 📉 Insights
- Identifies top-performing cryptocurrencies
- Shows fluctuations in crypto prices over time
- Helps understand market trends
